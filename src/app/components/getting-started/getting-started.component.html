Instalacion

Es necesario cumplir los siguientes requisitos:

{{item}}

Link para descargar Java 17

Lo primero será descargar Forge para Minecraft 1.16.5.

Para ello necesitamos la versión 36.2.8 de Forge.

Link de descarga de Forge


Una vez descargado, podemos empezar la instalación haciendo doble clic en el archivo que se ha descargado.

Debemos seleccionar la versión "Client" cuando el instalador nos pregunte cual queremos.

Cuando nos salga una ventana emergente, presionamos OK y estaremos listos para instalar mods.

Para instalar los mods necesitamos la carpeta "Mods" que se encuentra en el servidor de Discord.

Vamos a extraer la carpeta en la ruta donde tengamos instalado minecraft.

Puedes acceder a tu carpeta .minecraft poniendo %appdata% en el buscador de Windows.

Para instalar los mods opcionales, necesitamos la carpeta "Mods Opcionales" que se encuentra en el servidor de Discord.

Como en el paso anterior, vamos a extraer los mods en nuestra carpeta .minecraft/mods.

Puedes acceder a tu carpeta .minecraft poniendo %appdata% en el buscador de Windows.

Al no ser obligatorios, puedes introducir solo los mods que quieras.

Una vez finalizado el proceso, inicia minecraft (seleccionando Forge).

Cuando hayas llegado hasta aquí mencionanos en el canal #Entrada de Discord.
